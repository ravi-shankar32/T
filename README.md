#  I Love You Web App

A simple and romantic **interactive web application** built with **HTML**, **CSS**, and **JavaScript**.  
The project focuses on emotional interaction, smooth animations, and a clean visual experience.

---

##  Features

* Animated button interaction
* Romantic messages displayed dynamically
* Background music playback
* Fullscreen background image
* Randomized message positioning
* Smooth fade-in text animations

---

##  Technologies Used

* **HTML** – page structure
* **CSS** – styling and animations
* **JavaScript** – interactivity and dynamic behavior

---

##  Installation

Clone or download the repository:

    git clone https://github.com/ErkanSoftwareDeveloper/love-website.git
    cd love-website
No additional dependencies are required.

---

##  Usage

Open the application in your browser:

    open index.html

or simply double-click the `index.html` file.

Click the **My Heart** button to start the animation, music, and messages.
